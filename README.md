# studies_transport_cat

Проект: Транспортный справочник (каталог)

Получает и обрабатывает данные об остановках и маршрутах. По запросам выводит информацию о маршрутах, остановках.
Находится в процессе разработки. Планируется добавить возможность получать информацию из JSON, добавить визуализацию (с помощью SVG)

Необходимая версия языка C++17
